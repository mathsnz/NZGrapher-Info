School Folders

School folders allow teachers to load their own datasets onto NZGrapher and then share a link directly to that dataset to their students so the students don't need to individually load the dataset.

(VIDEO TO COME)

If you would like a folder set up for you school, please contact me.

If you do not want students to be able to download the dataset if you prefix it with SECURE (all in capitals) the download dataset option will be unavailable.
