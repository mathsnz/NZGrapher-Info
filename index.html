NZGrapher Support

NZGrapher is an online graphing tool specifically designed for supporting the teaching of the statistics in New Zealand. The idea behind NZGrapher was to create a web based statistical package that can run on any device, without an install. 

This Site contains supporting material for NZGrapher as well as links to the NZQA and TKI websites to help find the info for NCEA easily. Looking for info on the New Standards? Go to ncea.education.govt.nz.

To see the old materials that used to be on MathsNZ go to https://old.mathsnz.com and https://students.mathsnz.com. The resources there are no longer being maintained, but are provided as is, where is, if anyone still does want to use them.
